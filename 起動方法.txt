  NutriGuide - 栄養指導支援システム 起動方法
═══════════════════════════════════════════════════

【ローカルで使う場合（毎回）】
1. このフォルダを右クリック →「ターミナルで開く」
2. 以下のコマンドを入力して Enter:
   python serve.py
3. ブラウザが自動で開きます（http://localhost:8080）
4. 終了するには ターミナルで Ctrl+C

【Vercelにデプロイして使う場合（おすすめ）】
1. https://github.com でアカウントを作成
2. 「New repository」→ nutriguide という名前で作成
3. https://vercel.com で GitHub ログイン
4. 「New Project」→ nutriguide を選択
5. 「Deploy」を押すだけで公開完了！
   → https://nutriguide-xxxx.vercel.app のようなURLが発行されます

【Obsidianノート参照の手順】
1. Chrome または Edge でアプリを開く
2. 「Obsidianノート」メニューを選択
3. 「Vaultフォルダを開く」をクリック
4. D:\Vaults\Nutrition-Public\Nutrition-Public を選択
5. フォルダへのアクセスを許可する

【Claudeを使った栄養指導コメント生成】
1. クライアントの情報・食事記録を登録
2. 「レポート生成」→「プロンプトをコピー」
3. https://claude.ai を開いて貼り付け → 送信
4. 栄養指導コメントが生成されます

═══════════════════════════════════════════════════
  データはすべてブラウザ内（localStorage）に保存。
  外部サーバーへの送信は一切ありません。
═══════════════════════════════════════════════════
